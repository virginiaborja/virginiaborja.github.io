# virginiaborja.github.io 👋

My personal website, hosted on **GitHub Pages** — a hub for my resume, portfolio, and web-development services.

## 🌐 Live Site

**[virginiaborja.github.io](https://virginiaborja.github.io/)**

The landing page is a simple card menu that links out to the three sections below.

## ✨ Site Sections

### 📄 [Resume](https://virginiaborja.github.io/resume/) — `resume/`
My professional resume as an interactive web page: skills, experience (web design & AI development at Palm Beach Atlantic University, internships at Engajer AI and The Ritz-Carlton Dallas, and more), projects, education, semester-by-semester coursework, languages, and awards. Includes a responsive hamburger navigation for mobile.

### 🎨 [Portfolio](https://virginiaborja.github.io/portfolio/) — `portfolio/`
A more personal take on my background — skills, experience, education, and photo galleries (family, travel, food, sports, and more) that show who I am beyond the code.

### 💻 [Web Developer](https://virginiaborja.github.io/webdeveloper/) — `webdeveloper/`
My freelance web-development page: services offered, my five-step process (consultation → quote → design → development → launch & support), a timeline of 20+ client and university projects, and contact info.

> 🇺🇸🇧🇷 **Bilingual:** the Web Developer page is available in English (`index.html`) and Portuguese (`index-pt.html`), with a language switcher on the page.

## 🛠️ Built With

- **HTML5** — semantic, accessible markup
- **CSS3** — flexbox and grid layouts, timelines, card grids, responsive breakpoints (styles are embedded in some pages and in per-section `css/styles.css` files in others)
- **Vanilla JavaScript** — small touches like the mobile menu toggle
- **Google Fonts** (Poppins) and **GitHub Pages** for hosting

No frameworks, no build step — every page is plain static HTML you can open directly.

## 🚀 Running Locally

```bash
git clone https://github.com/virginiaborja/virginiaborja.github.io.git
cd virginiaborja.github.io
```

Then open `index.html` (or any section's `index.html`) in your browser.
